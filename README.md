## 👉 프로젝트 소개
- 게임 장르 : 4지선다 2D게임
- 플랫폼 : 웹&모바일
- 제작 인원 : 3 인 (기획1, 디자인1, 개발1)
- 제작 기간 : 각 2주 + QA 1주
- 사용 엔진과 언어 : Phaser, Python
- 작업물: 미션 제곱근의 사칙연산, 포물선 골프스타
- 특이사항: 몰랐던 언어를 단기간에 새로 공부해서 작업함
